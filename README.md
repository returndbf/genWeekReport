# genWeekReport
Generate a week report  for use git local logs


## Usage

### esm
```js
import { genWeekReport } from "genweekreport"
```

### cjs
```js
const { genWeekReport } = require("genweekreport")
```
 
### demo
```js
genWeekReport().then(report=>{
    console.log(report) // 1:sing2:jump3:rap4:basketball
})
```
